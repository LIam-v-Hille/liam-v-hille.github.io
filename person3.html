Although no as well known as the last two product. Red Hat inc.'s Fedora Linux is an amazing tool for coding and desktop computing in general. built off of the very popular Linux Kernal, Fedora will run well on almost anything. Red Hat also made a version for servers.

The main reasons you would want to use Fedora is because of its open source nature and the very powerful bash terminal.
